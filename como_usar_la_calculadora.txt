1-. Ejecute el programa de la calculadora.
2-. Introduzca el comando desdeado; puede indicar seguidamente los argumentos
    o proveerlos en otra línea, si corresponde.
3-. Una vez que tenga los resultados deseados, use el comando "regresar" para
    salir de la calculadora.

* Puede leer el archivo de configuración con el comando "verconfig"
* Puede ver el registro interno de errores con el comando "errores".
* Puede ver el historial de operaciones con el comando "historial".

* Además, puede ejecutar la calculadora desde el intérprete de línea de comandos
  de su sistema y darle de entrada el archivo de prueba para comprobar que no
  surjan errores inesperados irrecuperables.
